Programmation

Malgré ma découverte récente du monde de L'informatique et plus précisément de la Programmation, je suis capable d'utiliser de nombreux langages de programmation tels que le C, le Java, Python, voire même des langages utilisés pour la gestion de bases de données tels que SQL ou même Tutorial D. Même si je suis nouveau dans le domaine, la Programmation logicielle est pour l'instant ce qui me plaît le plus. Néanmoins, je reste ouvert à l'idée d'apprendre de nouveaux langages / nouvelles choses.

Développement web

Concernant le développement web, j'utilise majoritairement l'HTML et le CSS. Je Commence à apprendre le JavaScript, mais je ne fais que débuter dans ce domaine. Je n'avais Jamais utilisé ces langages avant mon arrivée à L'IUT de Lannion, et ce fut une Très Bonne Découverte. Même si je suis Aphantasique (dénué d'image mentale), je ne suis Pas le Meilleur dans tout ce qui est Design. Cependant, ces langages sont assez explicites pour parvenir à des Résultats assez Convenables malgré ce problème.

Environnement de développement

Concernant les environnements de développement, j'utilise majoritairement VSC pour tout ce qui est de la Programmation web mais je l'utilise aussi pour programmer en C et même parfois en Python. Pour ce qui est des autres IDE, j'utilise Eclipse pour programmer en Java ou alors en JavaFX. J'utilise aussi PostgreSQL pour tout ce qui a un rapport avec la création de bases de données ou alors l'utilisation du langage SQL. Je suis aussi capable d'utiliser Geany ou Spyder même si ce ne sont pas les IDE que j'utilise le plus.

Logiciels

Concernant la Programation (que ce soit logiciels ou bien web), je n'utilise Pas beaucoup de logiciels à part VisualParadigm qui me sert afin de réalisé des Diagramme UML pour l'utilisé dans du Java ou même pour réalisé une Base de donnée. Cependant, en ce qui concerne la Musique, ce n'est pas la même histoire. En effet, j'utilise des logiciels comme FlStudio afin de réalisé des musiques que vous pouvez écouter Ici.

Sites utilisés
