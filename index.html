Programation

Pour la partie programation, j'utilise de nombreux langages tel que le C, le Java, le Python, ou encore le PHP.

Développement web

Pour la partie développement web, J'utilise majoritairement de L'html et du css. Je commence à apprendre le javascript mais je ne fait que débuter dans ce domaine. Je n'avais jamais utilisé ces languages avant mon arriver à l'IUT de Lannion, et ce fut une très bonne découverte. Même si je suis aphantasyque (dénué d'image mental), je ne suis pas le meilleur dans tout ce qui est design. Cependant, ces langages sont assez explicites pour parvenir à des résultats assez convenable malgrés ce problème.

Environnement de développement

Concernant les environnements de développement, j'utilise majoritairement VSC pour tout ce qui est de la Programation web mais je l'utilise aussi pour programmer en C et même parfois en Python. Pour ce qui est des autres IDE, J'utilise Eclipse pour programmer en Java ou alors en JavaFx. J'utilise aussi PostgreSQL pour tout ce qui a un rapport avec la création de base de donnés ou alors l'utilisation du language SQL. Je suis aussi capable d'utilisé Geanny ou Spyder même ce ne sont pas les IDE que j'utilise le plus
